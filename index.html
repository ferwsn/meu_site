Início

Olá, seja bem-vindo ao meu site! Este é um dos vários projetos que pretendo realizar ao decorrer deste ano, tendo como objetivo principal o aprimoramento dos meus conhecimentos em desenvolvimento HTML, CSS e PHP.

Apesar de pretender ser um cientista de dados, eu acredito que o conhecimento em HTML é essencial por uma variedade de fatores:

  • O HTML, mesmo sendo uma linguagem de marcação, não de programação, serve como um ótimo início para a familiarização de conteúdos pertinentes à programação como um todo;

  • É útil como familiarização para IDEs e outras plataformas de desenvolvimento;

  • Serve para elucidar qual carreira profissional um desenvolvedor deve seguir: se será front-end (JavaScript) ou back-end (PHP, SQL);

  • O uso de HTML e CSS pode aprimorar consideravelmente a interface e o UX (user experience) de dashboards e outras plataformas de visualização de dados e Business Inteligence.
